# てり杯 / Teri-hai

> Web app for the Teri-hai tournament.

[🔗 Live Site](https://terihai.vercel.app)

## ✨ Features

- 🏆 Tournament bracket / results display
- 📡 RSS/XML feed parsing
- 📱 Responsive layout

## 🛠 Tech Stack

- Next.js + React + TypeScript
- Chakra UI
- fast-xml-parser

## 🚀 Development

```bash
pnpm install
pnpm dev
```
